## Instalación

```bash
$ git clone https://github.com/YormanAlexanderCarrillo/Backend_aplication_tutoring_soft_II.git
```
## Instalar modulos

```bash
$ npm install
```

## iniciar la aplicación

```bash
# Modo desarrollo
$ npm run start

# Modo escucha
$ npm run start:dev

# Modo Producción
$ npm run start:prod
```
